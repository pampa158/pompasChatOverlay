# Chatpompastv 📺
Chatpompastv es una aplicación de escritorio que muestra un overlay siempre visible en tu pantalla *(always on top)* para leer tu chat de Twitch en tiempo real.

## ✨ Características principales:
- 🖥️ Siempre visible: El chat se mantiene en primer plano sobre cualquier juego o ventana.
- 🔐 vinculación rápida: Colocá tu usuario de Twitch de forma rápida.
- 🌓 Opacidad ajustable: Baja la opacidad para que el chat no bloquee tu vista.
- ⚖️ Mínimo consumo: Si no fuera por el *Always on top* no te das cuenta que está abierto.

## ¿Cómo descargar y usar la app?
 ### NO requiere instalación.
  1. Descargá el archivo `Chatpompastv.exe` desde la lista de archivos de este repositorio o en el link.
     https://github.com/pampa158/pompasChatOverlay/releases/download/v1.0.0/Chatpompastv.exe
  3. Ejecutá el .exe con doble click.
  4. En configuración escribí tu "Usuario de Twitch" para que la app lo reconozca.
  5. ¡Listo! Ajusta la opacidad a tu gusto y leé a tu comunidad.

La idea a futuro es migrar el proyecto a tecnologías más modernas para dar mayor personalización visual.

```Al ejecutar por primera vez se genera un archivo "config.json" que guarda la info del ultimo inicio, no afecta al funcionamiento borrarlo solamente hay que volver a cargar el usuario de Twitch```
